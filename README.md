# sword-health-challenge

Frontend Software Engineer Challenge

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, install dependencies and run the development server:

```bash
npm install
```

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Next.js** with App Router and TypeScript
- **ESLint** for code quality and linting
- **Prettier** for consistent code formatting
- **CSS Modules** for component-level styling
- **Jest + React Testing Library** ready for unit and integration tests
- Optimized font loading via [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts)

## Project structure

```bash
├──src
    ├──/app           # Next.js App Router pages
    ├──/components    # Reusable React components
    ├──/types         # Global TS types
```

## Components

- `Header` - site navigation with active link highlighting
- `Footer` - fixed at the bottom of the page
- `PageContainer` - wraps pages with consistent structure and styling

## Available Scripts

- `npm run dev` - starts the development server
- `npm run build` - builds the app for production
- `npm start` - runs the production build
- `npm run lint` - runs ESLint to check code quality
- `npm run format` - runs Prettier to format code
- `npm run test` - runs unit tests
